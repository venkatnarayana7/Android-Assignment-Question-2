# Android Assignment Question 2 – Interactive Graphics using Canvas

## 📌 Overview

This project is developed as part of the **CSE227 – Advanced Android App Development** course assignment.

The application demonstrates Android's **Canvas API** by drawing multiple geometric shapes and applying animations to create an interactive graphics dashboard.

---

## 🎯 Features

- Draws geometric shapes using Canvas
  - Circle
  - Rectangle
  - Line
  - Triangle
- Smooth animation of moving objects
- Rotating graphical element
- Interactive touch event handling
- Center-aligned dashboard layout
- Score counter
- Clean and responsive UI

---

## 🛠️ Technologies Used

- Kotlin
- Android Studio
- Android Canvas API
- Custom View
- Paint
- Path
- MotionEvent
